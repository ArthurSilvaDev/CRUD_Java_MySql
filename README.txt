Este programa foi realizado em um exercicio de integração com JAVA e MySql, para operar é necessário existir um banco chamado "ideia", no usuário root e com senha 1234, meramente por questões de apresentação.

Para realizar o uso deste software no seu computador com um banco de dados específico, é necessário configurar a classe "ConexaoDAO" com as informações do seu banco de dados e alterar a tabela conforme informações necessárias.

Sistema desenvolvido por Arthur Felipe da Silva, da classe de Programação Orientada a Objetos II de 2023/2, do curso de Análise e Desenvolvimento de sistemas do IFRS - Campus Osório.


-----------------------------------
This program was carried out in an integration exercise with JAVA and MySql. To operate, it is necessary to have a database called “idea”, under the user called "root" and with password "1234", purely for presentation purposes.

To use this software on your computer with a specific database, you need to configure the “ConexaoDAO” class with your database information and change the table according to the necessary information.

System developed by Arthur Felipe da Silva, from the Object Oriented Programming II class of 2023/2, from the Analysis and Systems Development course at IFRS - Campus Osório.
